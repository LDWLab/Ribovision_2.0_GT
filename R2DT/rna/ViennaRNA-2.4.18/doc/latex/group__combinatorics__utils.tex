{\ttfamily \#include $<$\mbox{\hyperlink{combinatorics_8h}{Vienna\+RNA/combinatorics.\+h}}$>$}



Enumerate all necklaces with fixed content. 

This function implements {\itshape A fast algorithm to generate necklaces with fixed content} as published by Joe Sawada in 2003 \cite{sawada:2003}.

The function receives a list of counts (the elements on the necklace) for each type of object within a necklace. The list starts at index 0 and ends with an entry that has a count of 0. The algorithm then enumerates all non-\/cyclic permutations of the content, returned as a list of necklaces. This list, again, is zero-\/terminated, i.\+e. the last entry of the list is a {\ttfamily NULL} pointer.
