Create an annotated Post\+Script alignment plot. 

Similar to \mbox{\hyperlink{group__alignment__plots_ga3bb7409f5b73c2273f9f76cf8b754034}{vrna\+\_\+file\+\_\+\+PS\+\_\+aln()}} but allows the user to print a particular slice of the alignment by specifying a {\ttfamily start} and {\ttfamily end} position. The additional {\ttfamily offset} parameter allows for adjusting the alignment position ruler value.
