{\footnotesize\ttfamily typedef int() vrna\+\_\+callback\+\_\+pk\+\_\+plex\+\_\+score(const short $\ast$pt, int start\+\_\+5, int end\+\_\+5, int start\+\_\+3, int end\+\_\+3, void $\ast$data)}



{\ttfamily \#include $<$\mbox{\hyperlink{pk__plex_8h}{Vienna\+RNA/pk\+\_\+plex.\+h}}$>$}



Pseudoknot loop scoring function prototype. 

This function is used to evaluate a formed pseudoknot (PK) interaction in \mbox{\hyperlink{group__pseudoknots_ga2ac2baaedebca166f0000b7e61067a7c}{vrna\+\_\+pk\+\_\+plex()}}. It is supposed to take a PK-\/free secondary structure as input and coordinates of an additional interaction site. From this data, the energy (penalty) to score the PK loop is derived and returned in decakal/mol. Upon passing zero in any of the interaction site coordinates ({\ttfamily start\+\_\+5}, {\ttfamily end\+\_\+5}, {\ttfamily start\+\_\+3}, {\ttfamily end\+\_\+3}) or a {\itshape NULL} pointer in {\ttfamily pt}, the function must return a PK loop score. This minimum PK loop score is used in the first phase of the heuristic implemented in \mbox{\hyperlink{group__pseudoknots_ga2ac2baaedebca166f0000b7e61067a7c}{vrna\+\_\+pk\+\_\+plex()}} to assess whether a particular interaction is further taken into account in a later, more thorough evaluation step.

The simplest scoring function would simply return a constant score for any PK loop, no matter what type of loop is formed and how large the loop is. This is the default if \mbox{\hyperlink{group__pseudoknots_gaa6a5f5e20804ba729134eb8ad0c8da54}{vrna\+\_\+pk\+\_\+plex\+\_\+opt\+\_\+defaults()}} or \mbox{\hyperlink{group__pseudoknots_ga8f224a6aa666fd7444b225f676471992}{vrna\+\_\+pk\+\_\+plex\+\_\+opt()}} is used to generate options for \mbox{\hyperlink{group__pseudoknots_ga2ac2baaedebca166f0000b7e61067a7c}{vrna\+\_\+pk\+\_\+plex()}}.

{\ttfamily \#include $<$\mbox{\hyperlink{pk__plex_8h}{Vienna\+RNA/pk\+\_\+plex.\+h}}$>$}



Predict Pseudoknot interactions in terms of a two-\/step folding process. 

Computes simple pseudoknot interactions according to the PKplex algorithm. This simple heuristic first compiles a list of potential interaction sites that may form a pseudoknot. The resulting candidate interactions are then fixed and an PK-\/free MFE structure for the remainder of the sequence is computed.

The {\ttfamily accessibility} argument is a list of opening energies for potential interaction sites. It is used in the first step of the algorithm to identify potential interactions. Upon passing {\itshape NULL}, the opening energies are determined automatically based on the current model settings.

Depending on the {\ttfamily options}, the function can return the MFE (incl. PK loops) or suboptimal structures within an energy band around the MFE. The PK loop is internally scored by a scoring function that in the simplest cases assigns a constant value for each PK loop. More complicated scoring functions can be passed as well, see \mbox{\hyperlink{group__pseudoknots_ga7c413e55ee5117ef87193461241319c3}{vrna\+\_\+callback\+\_\+pk\+\_\+plex\+\_\+score}} and \mbox{\hyperlink{group__pseudoknots_ga2b0f46d082053d7543cf31237f32b433}{vrna\+\_\+pk\+\_\+plex\+\_\+opt\+\_\+fun()}}.

The function returns {\itshape NULL} on any error. Otherwise, a list of structures and interaction coordinates with corresponding energy contributions is returned. If no PK-\/interaction that satisfies the options is found, the list only consists of the PK-\/free MFE structure.
